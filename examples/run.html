  • How-to run the RMI sample program (Adder)


    0. Before you start, you need to create four files from lecture notes.

    Adder.java Interface Adder
    AdderImpl.java RMI Server Side
    AdderClient.java RMI Client Side
    policy Security Policy


    In Unix (Yallara/Numbat)

    1. Start RMIRegistry with a specified unique port number

    > rmiregistry 13759

    2. Compile your java files. (Remember to change the port number in your source code first.)
    # Change to the directory which contains the four required lab files.

    > javac *.java

    3. Start the RMI server side

    > java -Djava.security.policy=policy -Djava.rmi.server.codebase=file://home/u/user/RMILab/ AdderImpl
    # home/u/user should be replaced with your own home directory

    You shoud see:
    Created and registered AdderImpl object

    4. Start another xterm to execute the client side

    > java AdderClient

     

    In Windows Console

    1. Start RMIRegistry with a specified unique port number

    Start -> Run -> cmd
    H:\> start rmiregistry 13759

    2. Compile your java files. (Remember to change the port number in your source code first.)

    H:\> cd RMILab
    H:\RMILab> javac *.java
    # Change to the directory which contains the four required lab files.

    3. Start the RMI server side

    H:\RMILab> java -cp . -Djava.security.policy=policy -Djava.rmi.server.codebase=file:///H:/RMILab/ AdderImpl

    You shoud see:
    Created and registered AdderImpl object

    4. Start the RMI client side

    Start -> Run -> cmd
    H:\> cd RMILab
    H:\RMILab> java -cp . AdderClient

     

    In Eclipse

    1. Start RMIRegistry with a specified unique port number

    2. Create a new project and import four required files

    3. Put the following line in "VM arguments" of your "AdderImpl"
    running configuratoin (Run-Run...).

    Replace ### with your Eclipse Project Name

    -Djava.security.policy="file:/${workspace_loc}/###/policy"

    -Djava.rmi.server.codebase="file:/${workspace_loc}/###/bin/"

    Remeber to set the main class as "AdderImpl".

    4. Click the "Run" button.

    You shoud see :
    Created and registered AdderImpl object

    5. Open the AdderClient file, right click, Run as -> Java Applicatoin


    Alternately

    add the following line in your RMIserver:

    java.rmi.registry.LocateRegistry.createRegistry(port)
